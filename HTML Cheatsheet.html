Key HTML elements & attributes (15+)

  • <!DOCTYPE html>Defines the HTML5 document type. Example: <!DOCTYPE html>.
  • <html> langRoot element; include lang for accessibility (e.g., <html lang="en">).
  • <head>Metadata container (charset, viewport, title, links).
  • <meta> (charset/viewport)Essential meta tags, e.g. <meta charset="utf-8">, <meta name="viewport" content="width=device-width,initial-scale=1">.
  • <title>Page title shown in browser tabs and search results.
  • <link> (stylesheets)Include stylesheets, e.g. <link rel="stylesheet" href="Style.CSS">.
  • <script>JS include; prefer defer for external scripts.
  • <a> (links)Anchors for navigation. Use descriptive text and rel="noopener" for external links.
  • <img> (alt text)Must include alt for accessibility: <img src="photo.jpg" alt="Description">.
  • <form> & input typesCollect input. Example: <form><input type="email" required></form>.
  • <button> & accessibilityUse real <button> elements with accessible labels.
  • Semantic sectionsUse <section>, <article>, <aside> to structure content.
  • HeadingsKeep logical order (<h1>–<h6>) for outline/accessibility.
  • ARIA attributesUse attributes (e.g., role="status", aria-live) for dynamic content.
  • Data attributesUse data-* for JS hooks (e.g., <div data-id="123">).
  • Download & targetUse download on anchors for files; target="_blank" rel="noopener" for new tabs.
