Custom properties (variables)

  • --accent-rgbRGB triple used for accents and shadows, e.g. rgba(var(--accent-rgb),0.16).
  • --primary-top, --primary-bottomTheme colors used in gradients, e.g. linear-gradient(180deg, rgba(var(--primary-top),0.9), rgba(var(--primary-bottom),0.8)).
  • --p1-color, --p2-colorPlayer colors used in game discs (red/yellow).
  • --cell-sizeBoard cell sizing (adjust with media queries for small widths).

Layout & responsive patterns

  • Flexbox for centering and header layouts (display:flex, gap, align-items).
  • CSS Grid for the board and resume two-column layout (grid-template-columns).
  • Clamp() for responsive sizes: font-size: clamp(14px, 0.9vw + 12px, 18px).
  • Media queries used to reduce --cell-size on small widths.

Visual effects

  • GradientsLinear and radial gradients for backgrounds and hover states (linear-gradient(...), radial-gradient(...)).
  • 3D / elevationUse layered shadows and subtle transforms (box-shadow, transform: translateY()).
  • Frosted lookUse backdrop-filter: blur(..) saturate(..) on cards for depth.

Interaction & accessibility

  • :hover, :active, :focus and :focus-visible for keyboard focus outlines.
  • :focus-within used to style parent when child is focused.
  • :has() used (progressive enhancement) to apply stronger effects when children are hovered/focused.
  • Use of ARIA in markup alongside CSS for accessible status overlays.

Pseudo elements & helpers

  • ::before and ::after used for decorative thumbnails and inner rims (e.g., .resume::before, #board::after).
  • Inset shadows (inset) to create grooves / rim highlights inside circular cells.

Small examples

List item hover (lift + gradient)

.list ul li:hover {
	background: linear-gradient(90deg, rgba(var(--accent-rgb),0.88), rgba(109,13,109,0.88));
	transform: translateY(-6px);
	box-shadow: 0 18px 36px rgba(var(--accent-rgb),0.22);
}

Disc styling (glossy)

.disc.p1 {
	background: radial-gradient(circle at 30% 30%, rgba(255,255,255,0.45), var(--p1-color));
	box-shadow: 0 12px 28px rgba(0,0,0,0.28), inset 0 4px 12px rgba(255,255,255,0.06);
}

Responsive clamp

font-size: clamp(14px, 0.9vw + 12px, 18px);
