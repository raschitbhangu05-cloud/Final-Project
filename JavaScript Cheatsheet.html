Core JavaScript concepts (15+)

  • Variables(let, const, var): prefer let/const. Example: const x = 3;.
  • Data typesnumber, string, boolean, null, undefined, symbol, object.
  • Functions & declarationsdeclarations vs arrow functions; arrows don't bind their own this.
  • Scope & hoistingvar hoists; let/const are block-scoped.
  • Closuresfunctions that remember outer scope variables (useful for private state).
  • Objects & prototypesobject literals and prototype-based inheritance patterns.
  • Arrays & methodsmap, filter, reduce, forEach (e.g., [1,2,3].map(n => n*2)).
  • Async & Promisesfetch(url).then(r => r.json()).
  • Async/Awaitcleaner async code inside async functions (const res = await fetch(url)).
  • Fetch APIbrowser network requests (fetch('/api').then(res => res.json())).
  • DOM: selection & manipulation via document.querySelector, createElement, appendChild.
  • Events: el.addEventListener('click', handler); use event.preventDefault() for forms.
  • Higher-order functions: functions that accept or return functions (e.g., map).
  • Error handling: try/catch around async/await or sync code.
  • Modules: import/export for organization (e.g., import {x} from './x.js').
  • Helpers: JSON.parse/stringify, Object.keys, Array.from, setTimeout.
